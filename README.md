## 初始化脚本

这里通过脚本的方式抓取 LeetCode 题目信息，并初始化 Markdown 文档。

格式：`python3 init_contest.py <weekly/biweekly> <number>`

### Example

```
$ python3 init_contest.py weekly 155
```

输出：

```
saved 最小绝对差.
saved 丑数 III.
saved 交换字符串中的元素.
saved 项目管理.
```
